# ezsedの使ひ方



クリップボードの文書資料を正字正假名遣に變換します。



## 導入の仕方

以下の電子書類を同じフォルダに置きます。

- ezsed.py
- scriipt.sed

コマンド・プロンプトでPythonが使用できる必要があります。
installerでPthonを導入すれば、PATHは自動的に切られる筈です。

## 使用方法

1. テキストを選擇して複寫します(Ctrl-c)
2. コマンド・プロンプトでezsed.pyを實行します。
```コマンド・プロンプト
> cd \bin   ← ezsed.pyのあるディレクトリに移動する。
> python ezsed.py
```
3. 好きなエディタを起動します。
4. クリップボードのデータを貼付けます(Ctrl-v)
